# ZSH Terminal.app

A ZSH plugin for integrating with the new El Capitan Terminal.app features.

## Installation

Add the following to your `.zshrc`

```zsh
## Terminal.app specific title hooks
source ~/.zsh/zsh-terminal-app/terminal-app.plugin.zsh
```
